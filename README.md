# Sen Zhang - Personal Resume Website

A modern, responsive personal resume website featuring a dark theme with animated white grid frames and smooth scroll animations. Now structured as a multi-page website with dedicated pages for resume, work samples, and contact.

## 🌟 Features

- **Multi-Page Structure**: Separate pages for resume, work samples, and contact
- **Dark Theme Design**: Elegant dark background with cyan accent colors
- **Animated Grid Background**: Subtle moving grid pattern for visual appeal
- **Scroll Animations**: Elements animate in as you scroll down the page
- **Responsive Design**: Fully responsive across all devices
- **Interactive Elements**: Hover effects, smooth scrolling, and form validation
- **Modern UI/UX**: Clean, professional design with excellent user experience
- **Portfolio Showcase**: Dedicated work samples page with project cards
- **Advanced SEO**: Comprehensive search engine optimization with structured data
- **Social Media Ready**: Open Graph and Twitter Card meta tags for better sharing
- **Breadcrumb Navigation**: Improved site structure and user experience
- **Search Engine Friendly**: Sitemap, robots.txt, and semantic HTML structure

## 📋 Pages

### Landing Page (Resume) - `index.html`
1. **Hero Section**: Introduction with animated typing effect
2. **About**: Personal description with animated statistics
3. **Work Experience**: Timeline-based experience showcase
4. **Education**: Academic background with visual cards
5. **Publications**: Research and publication highlights
6. **Skills**: Interactive skill bars with progress animations

### Work Samples - `worksamples.html`
1. **Featured Projects**: Portfolio of recent work and achievements
2. **Project Cards**: Interactive cards showcasing different projects
3. **Technical Expertise**: Categorized skills and technologies
4. **Project Links**: Direct links to live demos and GitHub repositories

### Contact - `contact.html`
1. **Contact Information**: Email, phone, location, and social links
2. **Contact Form**: Interactive form for validation and professional inquiries
3. **Social Links**: Direct links to professional profiles

## 🛠️ Setup and Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/SenZhangResume.git
   cd SenZhangResume
   ```

2. Open `index.html` in your web browser

3. For local development, you can use a simple HTTP server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

## 🎯 Customization

### Personal Information
Update the following files:

**`index.html` (Resume Page)**:
- Name and title in the hero section
- About section content
- Work experience details
- Education information
- Publications
- Skills and proficiency levels

**`worksamples.html` (Work Samples Page)**:
- Project details and descriptions
- Technology tags for each project
- Project links (live demos, GitHub repositories)
- Technical expertise categories

**`contact.html` (Contact Page)**:
- Contact information (email, phone, location)
- Social media links
- Contact form settings

### Styling
Modify `styles.css` to customize:
- Color scheme (currently cyan theme)
- Typography
- Animations
- Layout spacing

### Functionality
Edit `script.js` to adjust:
- Animation timing
- Scroll behavior
- Form validation
- Interactive features
