  • EnneadTabLLM — Ollama-based local Retrieval Augmented Generation monorepo. Windows server (Ollama + RAG pipeline) + Vercel Next.js proxy + shared TypeScript types. Solves on-prem AI for HIPAA / data-sovereign clients.
  • AIA-Sustainable-Reporting — CRON service for Excel processing + frontend dashboard for AIA 2030 sustainable-design metrics.

  • EnneadTabHome — landing page and central hub at enneadtab.com.

EnneadTab
New York, NY | Started 2021, formal role since 2022
  • EnneadTab is a cross-application architectural plugin ecosystem that spans Rhino, Revit, CAD, Autodesk Desktop Connector, Enscape, InDesign, Photoshop, Excel, Email, Notification, AI, Computer Resource Monitoring, Text-to-Speech, and Sound Effects
  • It is a collection of 450+ quality-of-life tools for all types of daily workflows
  • Tailored toward office preferences and workflows at Ennead Architects
  • Resolves issues not yet addressed by official applications
  • Zero-learning-curve solution for designers, project architects, and BIM managers
  • This solo project started as a hobby in 2021 within a traditional project team and has evolved with increasing sophistication while staying current with office market preferences, digital infrastructure changes, and professional application API updates
  • An engaging and light-hearted support community helps increase adaptability within the office
  • Patented in the US
  • Development repo tests and compiles CPython as exe
  • Packaged and published to distribution repo
  • On the user side, a scheduled updater fetches content and updates the local codebase with delayed deletion to ensure silent background changes while the application is running
